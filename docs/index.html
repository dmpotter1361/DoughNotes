One container

Runs as a single Docker image. Clone, docker compose up, done.

Run it on your server

You need Docker and Git. That's it.

# 1. Get the code
git clone https://github.com/dmpotter1361/DoughNotes.git
cd DoughNotes

# 2. Set your session secret
cp .env.example .env
#    edit .env → set JWT_SECRET (e.g. `openssl rand -hex 32`)

# 3. Build & start
docker compose up -d --build

Then open http://<your-server>:3500 and register — the first account becomes the admin.
(Port 3500 is a nod to 350°F. 🔥)
